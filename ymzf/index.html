服务器部署说明

主控服务器 & 节点服务器 一键部署指南，含环境搭建、运行管理与状态监控

1

主控服务器运行

环境搭建

bash 
apt update -y && apt install g++ -y && apt install libssl-dev -y && apt install cron -y && systemctl enable --now cron

文件下载 + 修改权限

bash 
apt install -y wget && wget https://picgo91.cdn456.eu.org/ymzf/transfer_server && chmod 744 /root/transfer_server

运行

bash 
./transfer_server

查看是否启动成功

bash 
sudo systemctl status transfer_server

出现以下内容表示正常启动：

节点服务器运行

环境搭建 + 文件下载 + 修改权限

bash 
apt update && apt install -y g++ libssl-dev && apt install -y libnetfilter-queue1 && apt install -y wget && wget https://picgo91.cdn456.eu.org/ymzf/redirect_server_https && chmod 744 /root/redirect_server_https

查看主控后台的节点安装命令

比如后台命令是：./redirect_server_https -s wwvip.ap220.com -t 443 --master-ip 52.192.28.35 --node-name "胖子"

常用管理命令汇总

主控服务器 (transfer_server)

停止服务
sudo systemctl stop transfer_server.service
停止自启动
sudo systemctl disable transfer_server.service
删除服务文件
sudo rm /etc/systemd/system/transfer_server.service
删除程序
sudo rm /root/transfer_server
查看服务状态
sudo systemctl status transfer_server

节点服务器 (redirect_server)

停止服务
sudo systemctl stop redirect_server.service
停止自启动
sudo systemctl disable redirect_server.service
删除服务文件
sudo rm /etc/systemd/system/redirect_server.service
删除程序
sudo rm /root/redirect_server_https
查看服务状态
sudo systemctl status redirect_server
